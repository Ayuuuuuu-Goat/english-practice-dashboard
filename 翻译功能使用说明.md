# AI 对话翻译功能 - 使用说明

## ✅ 新功能上线！

现在每条对话和每个选项都可以**一键翻译成中文**！

## 🌟 功能特点

### 1. **消息翻译**
- AI 的对话可以翻译
- 你的回复也可以翻译
- 点击图标即可查看中文

### 2. **选项翻译**
- 每个回复选项都可以翻译
- 看不懂英文？先翻译再选择
- 帮助理解选项含义

### 3. **智能缓存**
- 翻译一次，永久保存
- 再次点击切换显示/隐藏
- 不重复调用 API

## 🎮 如何使用

### 步骤 1: 刷新浏览器
```
Cmd+Shift+R (Mac)
Ctrl+Shift+R (Windows)
```

### 步骤 2: 开始对话
进入 "AI 场景对话"，选择任意场景

### 步骤 3: 翻译消息
1. 在每条消息右上角，有一个 **🌐 翻译图标**
2. 点击图标，等待 1-2 秒
3. 中文翻译会显示在消息下方
4. 再次点击可以隐藏翻译

### 步骤 4: 翻译选项
1. 在每个回复选项右侧，也有 **🌐 翻译图标**
2. 点击查看该选项的中文翻译
3. 理解后再做选择

## 🎯 翻译按钮位置

### 对话消息
- **位置**: 消息右上角
- **图标**: 🌐 (Languages icon)
- **状态**:
  - 未翻译: 灰色图标
  - 翻译中: 旋转加载图标
  - 已翻译: 图标（再点击隐藏）

### 回复选项
- **位置**: 选项右侧
- **图标**: 🌐 (Languages icon)
- **功能**: 点击查看翻译，再点击隐藏

## 📋 常见问题

### Q1: 翻译按钮在哪里？
**A**:
- 消息：右上角的 🌐 图标
- 选项：右侧的 🌐 图标

### Q2: 可以自动翻译所有内容吗？
**A**: 目前需要手动点击。这样设计是为了：
- 节省 API 成本
- 鼓励先尝试理解英文
- 按需加载，更灵活

### Q3: 翻译会保存吗？
**A**: 会！同一个会话中，翻译过的内容会缓存，再次点击直接显示，不会重复翻译。

### Q4: 翻译会影响评分吗？
**A**: 不会！翻译只是辅助理解，不影响对话评分。

### Q5: 可以翻译成其他语言吗？
**A**: 目前只支持中文。如需其他语言，请反馈。

### Q6: 翻译速度慢怎么办？
**A**:
- 正常是 1-2 秒
- 如果超过 5 秒，可能是网络问题
- 刷新页面重试
