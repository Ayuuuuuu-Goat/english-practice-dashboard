# 技术播客精选功能 - 使用说明

## ✨ 新功能上线！

## 🎮 如何使用

### 1. 进入播客精选
- 点击左侧边栏的 **"技术播客精选"** 图标（耳机图标）
- 首次加载会显示示例播客

### 2. 选择播客
- 左侧列表显示所有可用播客
- 查看分类标签：AI/科技、创业故事、产品设计、领导力
- 查看难度级别和时长
- 点击播客卡片切换

### 3. 收听播客
**收听标签页：**

#### 播放控制
```
┌─────────────────────────────────────────┐
│     [进度条滑块]                         │
│     0:25 / 7:00                          │
│                                          │
│     ⏪   ▶️   ⏩                          │
│    10秒  播放  10秒                      │
│                                          │
│     🔊 [音量滑块] 80%                    │
```

**操作说明：**
- 点击播放按钮开始/暂停
- 拖动进度条跳转到指定位置
- 快退/快进按钮跳过 10 秒
- 调节音量滑块控制音量

#### 文本对照
- 播放器下方显示完整演讲稿
- 清晰的段落分隔
- 便于跟随理解

### 4. 学习生词
**生词标签页：**

每个生词卡片包含：
- 📌 序号编号
- 📌 单词 + 音标
- 📌 英文释义
- 📌 中文释义
- 📌 上下文例句（来自播客）
- 📌 时间戳按钮（点击跳转到该词出现位置）
- 📌 难度标签

**使用技巧：**
1. 先看英文释义，尝试理解
2. 查看中文释义确认
3. 阅读上下文加深印象
4. 点击时间戳回到播客中听发音

### 5. 听写练习
**听写标签页：**

#### 练习流程
```
1️⃣ 点击"播放片段"按钮
   ↓
2️⃣ 仔细听音频（可重复播放）
   ↓
3️⃣ 在文本框输入听到的内容
   ↓
4️⃣ 点击"提交答案"
   ↓
5️⃣ 查看准确度评分和正确答案
   ↓
6️⃣ 点击"下一个片段"继续练习
```

#### 评分系统
- ✅ **90-100%**：优秀！听力和拼写都很准确
- ✅ **70-89%**：良好，大部分内容正确
- ⚠️ **50-69%**：及格，需要加强练习
- ❌ **< 50%**：需要重复听多次，注意细节

## 💡 学习建议

### 最佳学习流程
```
第一遍：完整收听（不看文本）
   ↓
第二遍：对照文本收听
   ↓
第三遍：学习生词
   ↓
第四遍：完成听写练习
   ↓
第五遍：复习巩固
```

### 听力技巧

#### 初次收听
- 🎯 **目标**：理解大意
- 🎯 **方法**：不要纠结每个词，抓住关键信息
- 🎯 **笔记**：记录主要观点

#### 对照文本
- 📖 **标注**：标记不熟悉的词汇
- 📖 **理解**：搞清楚不懂的句子结构
- 📖 **发音**：注意连读、弱读现象

#### 听写练习
- ✍️ **重复**：多听几遍再写
- ✍️ **完整**：尽量写出完整句子
- ✍️ **细节**：注意时态、单复数、冠词等

### 难度选择

**初级（Easy）：**
- 语速较慢
- 词汇简单
- 句子结构简单
- 建议：新手从这里开始

**中级（Medium）：**
- 正常语速
- 有一些专业术语
- 句子结构中等复杂
- 建议：有一定基础后挑战

**高级（Hard）：**
- 快速语速
- 大量专业术语
- 复杂句式和表达
- 建议：进阶学习者

## ❓ 常见问题

### Q1: 为什么听不到声音？
**A**:
- 检查音量设置（播放器音量和系统音量）
- 确认音频 URL 可访问
- 检查浏览器是否阻止自动播放

### Q2: 进度条不能拖动？
**A**: 这是正常的，音频需要先加载元数据。等待几秒后再尝试。

### Q3: 听写准确度太低怎么办？
**A**:
- 多听几遍再写
- 注意大小写和标点
- 从简单片段开始练习
- 对照答案学习正确写法

### Q4: 可以下载音频吗？
**A**: 目前不支持下载，需要在线收听。

### Q5: 生词可以导出吗？
**A**: 目前暂不支持，可以手动记录到笔记本。未来可能添加生词本功能。
