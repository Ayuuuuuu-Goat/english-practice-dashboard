# AI 场景对话 - 使用指南

## ✅ 功能已就绪！

所有代码已经部署完成，服务器正在运行在 http://localhost:3000

## 🚀 现在就开始使用

### 步骤 1: 刷新浏览器页面
在浏览器中按 **Cmd+Shift+R** (Mac) 或 **Ctrl+Shift+R** (Windows) 进行硬刷新

### 步骤 2: 点击 "AI 场景对话" 标签
在侧边栏中找到并点击 "💬 AI 场景对话"

### 步骤 3: 选择一个场景
你会看到 4 个场景卡片：
- 🎯 **Technical Interview** (困难)
- 👔 **Team Meeting Discussion** (中等)
- 📊 **Project Status Report** (中等)
- 🤝 **Client Negotiation** (困难)

点击任何一个开始对话！

### 步骤 4: 进行对话
1. AI 会扮演角色（面试官、老板、客户等）开始对话
2. 你会看到 2-3 个对话选项
3. **点击选项**（不用打字！）
4. 立即看到反馈：
   - ✅ 质量评分
   - 📝 语法建议
   - 💡 更好的表达
   - 📈 对剧情的影响

### 步骤 5: 触发结局
根据你的选择质量，会触发不同结局：
- 🏆 **最佳结局**: 获得Offer / 晋升 / 加薪 / 签单
- ✅ **普通结局**: 表现还行
- ❌ **失败结局**: 需要改进

## 🐛 如果遇到问题

### 问题1: 页面空白或加载失败
**解决方法**:
1. 硬刷新浏览器 (Cmd+Shift+R)
2. 清除浏览器缓存
3. 检查浏览器控制台 (F12) 是否有错误

### 问题2: 点击场景后没有对话内容
**解决方法**:
1. 确保你已经登录账号
2. 打开浏览器控制台 (F12) 查看错误信息
3. 检查服务器日志: `tail -f /tmp/nextjs-dev.log`

### 问题3: 显示 "Failed to create progress"
**解决方法**:
这是正常的，说明你使用的是测试环境。在浏览器中使用真实登录用户时不会有这个问题。

## 🧹 可选：清理重复数据

如果你发现场景列表中有重复项，在 Supabase SQL Editor 中运行：
```sql
scripts/cleanup-duplicate-scenarios.sql
```

这会删除重复的场景，只保留最新版本。

## 📊 统计数据

完成对话后，主页面会显示：
- 📊 总对话次数
- ✅ 完成对话次数
- 📈 平均得分
- 🏆 获得最佳结局次数

## 🎯 提高分数的技巧

1. **选择具体的回答** - 提供细节和数据
2. **使用专业表达** - 避免"I think maybe"
3. **展现自信** - 使用肯定的语气
4. **提供数据支持** - 用数字证明你的成果

- **第二遍**: 尝试不同选项，学习更好的表达
- **第三遍**: 应用学到的知识，争取最佳结局
- **实战应用**: 在真实工作中使用学到的表达！
