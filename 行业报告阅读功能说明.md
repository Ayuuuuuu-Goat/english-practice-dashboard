# 行业报告阅读功能 - 使用说明

## ✨ 新功能上线！

专业的英语行业报告阅读训练模块，帮助你提升专业英语阅读和写作能力。

- AI 评价反馈
- 批判性思维训练

## 📦 安装步骤

### 步骤 1: 创建数据库表

在 Supabase 项目中执行以下 SQL 脚本：

```bash
# 方法 1: 通过 Supabase Dashboard
1. 登录 Supabase Dashboard
2. 进入项目的 SQL Editor
3. 复制粘贴 scripts/create_industry_reports_tables.sql 的内容
4. 点击 Run 执行

# 方法 2: 通过命令行（如果配置了 Supabase CLI）
supabase db push
```

脚本位置：`scripts/create_industry_reports_tables.sql`

### 步骤 2: 验证表创建

执行后，你应该看到以下表被创建：
- `industry_reports` - 报告主表
- `report_vocabulary` - 词汇表
- `report_discussion_questions` - 讨论问题表
- `user_report_progress` - 用户进度表
- `user_report_summaries` - 用户摘要表
- `user_discussion_answers` - 用户回答表

## 🎮 如何使用

### 1. 进入报告阅读
- 点击左侧边栏的 **"行业报告阅读"** 图标（文档图标）
- 首次加载会显示示例报告

### 2. 选择报告
- 左侧列表显示所有可用报告
- 点击报告卡片切换
- 查看分类标签（AI/科技/商业）
- 查看难度级别和预估时间

### 3. 阅读报告
**阅读标签页：**
- 完整的报告内容
- 清晰的章节结构
- 专业的排版格式

### 4. 学习词汇
**词汇标签页：**
- 10个重点专业词汇
- 包含音标、释义、例句
- 按难度分级
- 序号编排便于记忆

### 5. 撰写摘要
**摘要标签页：**
1. 用英文撰写报告摘要（建议 150-250 词）
2. 系统实时显示字数
3. 点击"提交摘要"
4. AI 评分（0-100分）
5. 收到详细反馈：
   - 优点分析
   - 改进建议
   - 语法问题
   - 整体评价

### 6. 回答讨论问题
**讨论标签页：**
1. 阅读 4 个思考问题
2. 用英文输入你的答案
3. 可以查看参考答案
4. 提交后获得 AI 评分和反馈

## 💡 使用技巧

### 最佳学习流程
```
1️⃣ 阅读报告（15分钟）
   ↓
2️⃣ 学习词汇（10分钟）
   ↓
3️⃣ 撰写摘要（15分钟）
   ↓
4️⃣ 回答问题（20分钟）
```

### 摘要写作建议
- **开头**：简要说明报告主题
- **主体**：概括 3-5 个关键要点
- **结尾**：总结报告的主要结论或影响
- **长度**：控制在 150-250 词
- **语言**：使用清晰、简洁的学术英语

### 讨论回答技巧
- **观点明确**：清楚表达你的立场
- **论据充分**：提供具体例子或数据支持
- **逻辑清晰**：使用 First, Second, Finally 等连接词
- **深度思考**：展示批判性思维能力
- **语言正式**：使用正式的学术语言

## ❓ 常见问题

### Q1: 为什么看不到报告列表？
**A**: 检查数据库表是否正确创建，并且示例数据已插入。

### Q2: 提交摘要后没有反馈？
**A**:
- 检查 API 密钥是否配置正确
- 查看浏览器控制台是否有错误
- 确认网络连接正常

### Q3: 可以删除或编辑已提交的摘要吗？
**A**: 目前不支持删除，但可以重新提交，系统会更新为最新版本。

### Q4: AI 评分的标准是什么？
**A**:
- 准确性（25分）
- 完整性（25分）
- 简洁性（20分）
- 语言质量（20分）
- 连贯性（10分）

### Q5: 如何查看历史学习记录？
**A**: 目前可以通过数据库查询 `user_report_summaries` 和 `user_discussion_answers` 表查看。未来可能添加学习历史页面。
